## ğŸš€ Getting Started :

### Prerequisites :
- A modern web browser (Chrome, Firefox, Safari, Edge)
- A text editor or IDE for modifications

### Installation :

1. **Clone or download** the project files to your local machine
2. **Ensure all files** are in the same directory structure as shown above
3. **Add the missing CSS file** (`style.css`) as it's referenced but not included
4. **Add images** to the `IMG/` directory with the filenames referenced in the HTML
5. **Open `index.html`** in your web browser

### Running the Website :

Simply open `index.html` in any modern web browser. The website is built with vanilla HTML, CSS, and JavaScript, so no server setup is required for basic viewing.
