# Rthilaa High School Website :

A modern, responsive website for Rthilaa High School serves as the digital front door to the institution, offering comprehensive information about the school’s academics, faculty, activities, and community. It is designed to provide prospective students, current families, and community members with essential details about our educational programs and vibrant school life.

## 🎯 Live Demo : 

🔗 [Rthilaa High School](https://rthilaa-high-school.netlify.app/)

We would appreciate it if you decide to use this project. Please include credit when using it. Thank you! 🙏

## ✨ Features :

- **Responsive Design**: Fully responsive layout that works seamlessly across desktop, tablet, and mobile devices
- **Interactive Navigation**: Smooth scrolling navigation with mobile hamburger menu
- **Dynamic Content**: Expandable sections for events, faculty, and student reviews
- **Modern UI**: Clean, professional design with engaging visual elements
- **Contact Forms**: Newsletter subscription and contact forms for easy communication
- **Social Media Integration**: Links to school's social media presence

## 🏗️ Structure :

### Sections Include:
- **Hero Section**: Welcome message and call-to-action buttons
- **About**: School mission, achievements, community stats, and campus features
- **Academics**: STEM, Arts, Humanities, Athletics programs, and AP courses
- **Activities**: Student clubs, athletics, and performing arts
- **Events**: Upcoming school events with dates and details
- **Faculty**: Meet our dedicated teaching staff
- **Student Reviews**: Testimonials from current students
- **Newsletter**: Email subscription for updates
- **Contact**: Contact information and message form

## 🛠️ Technologies Used :

- **HTML5**: Semantic markup for better accessibility and SEO
- **CSS3**: Modern styling with Flexbox and responsive design
- **Vanilla JavaScript**: Interactive functionality and DOM manipulation
- **Font Awesome**: Icon library for visual elements
- **Google Fonts**: Typography enhancement

## 🚀 Getting Started :

### Prerequisites :
- A modern web browser (Chrome, Firefox, Safari, Edge)
- A text editor or IDE for modifications

### Installation :

1. **Clone or download** the project files to your local machine
2. **Ensure all files** are in the same directory structure as shown above
3. **Add the missing CSS file** (`style.css`) as it's referenced but not included
4. **Add images** to the `IMG/` directory with the filenames referenced in the HTML
5. **Open `index.html`** in your web browser

### Running the Website :

Simply open `index.html` in any modern web browser. The website is built with vanilla HTML, CSS, and JavaScript, so no server setup is required for basic viewing.

## 🎨 Customization :

### Updating School Information :
- Modify content directly in `index.html`
- Update school name, contact information, and descriptions
- Replace placeholder images with actual school photos

### Styling Changes :
- Edit the CSS file (needs to be created as `style.css`)
- Customize colors, fonts, and layout to match school branding
- Modify responsive breakpoints as needed

### Adding Functionality :
- Extend `script.js` for additional interactive features
- Add form validation or submission handling
- Implement additional dynamic content sections

## 📱 Responsive Design :

The website is designed to be fully responsive with:
- **Mobile-first approach**: Optimized for mobile devices
- **Flexible layouts**: Adapts to different screen sizes
- **Touch-friendly**: Appropriate button sizes and spacing
- **Readable typography**: Scales appropriately across devices

## 🔧 Interactive Features :

### JavaScript Functionality:
- **Mobile Navigation**: Hamburger menu toggle for mobile devices
- **Expandable Sections**: Show/hide additional content for events, faculty, and reviews
- **Smooth User Experience**: Dynamic button text updates based on content state

### Expandable Content:
- **Events**: Toggle between showing 3 or 6 upcoming events
- **Faculty**: Display 4 or 8 faculty member profiles
- **Reviews**: Show 4 or 8 student testimonials
